# 과제 1: PyTorch 튜토리얼

## 환경 설정 (uv) + Jupyter

### 1) 가상 환경 생성

어떤 패키지 관리자를 사용해도 좋습니다. 여기서는 매우 빠른 Python 패키지 설치 및 관리자인 `uv`(https://github.com/astral-sh/uv)를 사용한 설정 방법을 설명합니다.

리포지토리 루트 디렉토리에서 다음을 실행합니다:

```bash
# Python 3.12로 가상 환경 생성
uv venv --python 3.12

# 환경 활성화
# macOS/Linux:
source .venv/bin/activate
# Windows:
# .venv\Scripts\activate

# 의존성 설치
uv pip install torch torchvision jupyter
```

Jupyter Notebook 사용법에 대한 소개는 다음 자료를 참고하시기 바랍니다: https://docs.jupyter.org/en/latest/
