## 환경 설정 (uv) + Jupyter

### 1) 가상 환경 생성

어떤 패키지 관리자를 사용해도 좋습니다. 여기서는 매우 빠른 Python 패키지 설치 및 관리자인 `uv`(https://github.com/astral-sh/uv)를 사용한 설정 방법을 설명합니다.

리포지토리 루트 디렉토리에서 다음을 실행합니다:

```bash

uv venv --python 3.12

# 환경 활성화
# macOS/Linux:
source .venv/bin/activate
# Windows:
# .venv\Scripts\activate

# 의존성 설치
uv pip install torch torchvision jupyter
```

Jupyter Notebook 사용법에 대한 소개는 다음 자료를 참고하시기 바랍니다: https://docs.jupyter.org/en/latest/

### 2. 과제 제출 (Assignment Submission)

여러분은 **"과제 1"**이라는 단일 과제에 **모든 제출물**(코드 및 비디오)을 제출하게 됩니다.

1. Gradescope의 과목 페이지로 이동합니다.

3. 컴퓨터에서 다음 파일들을 드래그 앤 드롭하거나 선택합니다:
